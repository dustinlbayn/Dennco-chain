# Dennco Chain Wallet

The wallet module will provide enterprise wallet operations for Dennco Chain.

## Planned capabilities

- Custodial and non-custodial account support
- Address book tied to identity registry records
- Transaction signing policies
- Role-based access controls
- Hardware/security module integration path
- Internal enterprise approval workflow

## Current status

This folder is a roadmap placeholder. The initial API gateway already exposes wallet balance lookup through `/wallet/:address/balance`.
